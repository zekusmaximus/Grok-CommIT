# Grok-CommIT Legacy Archive (v3.1)

This directory contains the archived codebase for Grok-CommIT v3.1 and earlier.

## Contents
- `summon.py`: The original CLI summoning engine.
- `api.py`: The original FastAPI wrapper.
- `web/`: The original HTML/JS web interface.
- `primers/`: Old system prompts.
- `vscode-extension/`: The deprecated VSCode extension.

**Note**: These files are kept for historical reference and lineage tracking. The active project is now located in `/backend` and `/frontend`.
